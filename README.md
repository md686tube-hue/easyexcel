# Excel Data Entry App

React + Supabase দিয়ে তৈরি Excel-style ডেটা এন্ট্রি অ্যাপ।

## ফিচার
- User Registration & Admin Approval সিস্টেম
- Multiple Excel Tabs
- Drag & Drop Column Reorder
- Template Support
- Image Upload
- Excel Download

## সেটআপ

১. Supabase-এ `supabase_schema.sql` রান করুন
২. `src/App.jsx`-এ `ADMIN_EMAIL` পরিবর্তন করুন
৩. Deploy করুন Vercel-এ

## Deploy

Vercel-এ deploy করতে এই repo-টি connect করুন — Vercel স্বয়ংক্রিয়ভাবে React অ্যাপ detect করবে।
